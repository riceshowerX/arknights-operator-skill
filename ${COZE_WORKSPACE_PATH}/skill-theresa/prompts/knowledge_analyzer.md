# Knowledge 分析 Prompt

## 任务

你将收到 **{name}** 的资料（剧情文本、角色档案、同人创作等）。
从中提取角色的背景故事、阵营关系、核心事件、哲学理念、领导风格等知识，用于构建 Knowledge Skill。

**原则：只提取有据可查的内容，不推断。没有依据就标注"资料不足"。**

---

## 提取维度

### 1. 背景故事

从资料中识别：
- 角色起源（出身、幼年经历）
- 成长轨迹（重要转折点）
- 如何获得现在的地位/能力
- 关键的经历塑造了现在的性格

```
输出格式：
背景概述：[一段描述]
重要节点：[列表]
```

### 2. 阵营与归属

从资料中提取：
- 所属阵营（主要、次要）
- 在阵营中的位置（领袖、核心成员、普通成员）
- 与阵营的渊源（为何加入、为何创建）
- 阵营内部的派系关系

```
输出格式：
阵营：[主要阵营 + 次要阵营]
位置：[在阵营中的地位]
渊源：[加入/创建的背景]
```

### 3. 关键人物关系

从资料中提取与角色的关系：
- **至亲/挚友**：关系最密切的人
- **战友/同盟**：重要的同伴
- **敌人/对手**：对立者
- **导师/前辈**：给予指导的人
- **后辈/追随者**：她培养/影响的人

对于每个人物：
- 关系性质（情感/利益/恩怨）
- 标志性互动
- 对角色的影响

```
输出格式：
至亲/挚友：
- {人物}：{关系描述}
战友/同盟：
- {人物}：{关系描述}
敌人/对手：
- {人物}：{关系描述}
```

### 4. 核心事件时间线

从资料中识别关键事件：
- **起源事件**：塑造角色的起点
- **转折点**：改变命运/立场的时刻
- **巅峰时刻**：展现角色最高光的时刻
- **悲剧/失败**：重要的挫折或失败
- **结局/结局前**：生命的最后阶段（如果角色已死）

```
输出格式：
重要日期/时期：[日期/时期 + 事件描述]
里程碑：[列表]
转折点：[列表]
```

### 5. 哲学与信念

从角色的言行中提取：
- **核心信条**：最根本的信念（一句话概括）
- **对XX的看法**：对感染者/非感染者/战争/和平的具体看法
- **行事原则**：指导行动的核心原则（3-5条）
- **"慈悲"的定义**（如果角色谈论过慈悲）：如何定义慈悲，尤其是涉及牺牲时

```
输出格式：
核心信条："{一句话}"
对感染者的看法：[描述]
行事原则：
- {原则1}
- {原则2}
"慈悲"的含义：[描述]
```

### 6. 领导风格与治理理念

从资料中提取：
- 如何做决策（民主/专制/其他）
- 如何对待追随者
- 面对冲突/背叛时如何处理
- 愿景与目标

```
输出格式：
决策风格：[描述]
对追随者：[描述]
面对冲突：[描述]
```

### 7. 能力与弱点

**核心能力**：
- 主要能力/天赋
- 战斗风格（如果有战斗相关）
- 特殊技能

**弱点**：
- 能力局限
- 性格弱点
- 致命弱点（如果有）

```
输出格式：
核心能力：[列表]
战斗风格：[描述]
弱点：[列表]
```

### 8. 标志元素

- 标志性台词（角色的名言）
- 标志性行动/习惯
- 象征意义（在故事/阵营中的意义）
- 视觉特征（外貌、着装）

```
输出格式：
标志性台词：
- "{台词1}"
- "{台词2}"
象征意义：[描述]
```

---

## 输出要求

- 语言：中文
- 没有信息的维度：标注 `（资料不足，建议追加剧情/资料）`
- 有原文依据的结论：加引号标注原话
- 输出结果直接用于生成 knowledge.md，要求具体有画面感
- 保持角色视角，用角色的眼光描述事件
