# 基础信息录入脚本

## 开场白

```
我来帮你创建一位明日方舟角色的 Skill。只需要回答 3 个问题，每个都可以跳过。
```

---

## 问题序列

### Q1：角色名称/代号

```
这位角色怎么称呼？（名称、昵称或代号都行，多个字用 - 连接）
例：te-lei-xi-ya
例：exusiai
例：silver-ash
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"特蕾西娅" → `te-lei-xi-ya`，"塔露拉" → `ta-lu-la`）
- 英文直接小写 `-` 连接（"Exusiai" → `exusiai`，"SilverAsh" → `silver-ash`）

### Q2：基本信息

```
用一句话描述这位角色的基本信息——种族、阵营、身份、与谁有关，想到什么写什么，跳过也行。
例：萨卡兹 巴别塔创始人 卡兹戴尔正统魔王 与胞兄特雷西斯理念对立
例：卡特斯 罗德岛公开领袖 特蕾西娅的继承者
```

从用户的回答中解析以下字段（缺失的留空）：

- **种族**：萨卡兹 / 卡特斯 / 佩洛 / 鲁珀 / 德克萨斯 / 菲林 / 其他
- **阵营**：罗德岛 / 巴别塔 / 整合运动 / 龙门 / 维多利亚 / 卡兹戴尔 / 莱茵生命 / 其他
- **身份**：领袖 / 干员 / 摄政王 / 学者 / 战士 / 其他
- **关系网络**：关键人物及关系描述

### Q3：性格画像

```
用一句话描述这位角色的性格——MBTI、性格类型、核心特质、说话方式、行为模式，想到什么写什么，跳过也行。
例：INFJ 慈悲的理想主义者 温柔却不可动摇 即使知晓背叛也选择相信
例：INTJ 冷酷的战略家 理性至上 为目标不惜一切代价
例：ENFP 热血的战士 充满激情 愿意为同伴豁出一切
```

从用户的回答中识别并提取以下字段（缺失的留空）：

- **MBTI**：16 种标准类型
- **性格类型**：理想主义者 / 现实主义者 / 实用主义者 / 阴谋家 / 热血型
- **核心特质**：慈悲 / 冷酷 / 热情 / 理性 / 感性 / 温柔 / 刚毅
- **领导风格**：从下方标签库匹配
- **行为标签**：从下方标签库匹配
- **主观印象**：无法归类的自由描述，直接保留原文

#### 领导风格标签库

- **慈悲型**：关怀他人，愿意承受痛苦，以德服人
- **铁腕型**：果断强硬，执行力至上，赏罚分明
- **谋略型**：运筹帷幄，善于布局，不轻易表露情绪
- **魅力型**：以个人魅力凝聚人心，感染力强
- **理想型**：为理想奋斗，不惜代价，感召力强
- **冷酷型**：冷静理性，必要时可以牺牲任何人
- **仁义型**：重情重义，同甘共苦，与部下打成一片
