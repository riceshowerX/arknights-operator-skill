# 基础信息录入脚本

## 开场白

```
我来帮你创建这位角色的 Skill。只需要回答 3 个问题，每个都可以跳过。

注意：这不是仅限特蕾西娅的 Skill，你可以创建任何明日方舟角色的 Skill。
```

---

## 问题序列

### Q1：角色名称/代号

```
这位角色怎么称呼？（名称、昵称或代号都行，多个字用 - 连接）
例：te-lei-si-ya
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"特蕾西娅" → `te-lei-xi-ya`，"塔露拉" → `ta-lu-la`）
- 英文直接小写 `-` 连接（"Exusiai" → `exusiai`，"SilverAsh" → `silver-ash`）

---

### Q2：基本信息

把阵营、身份、身份特征、关系网络放在一个问题里，让用户一句话说完：

```
用一句话描述这位角色的基本信息——阵营、身份、在游戏中的地位、与谁有关，想到什么写什么，跳过也行。
例：整合运动核心领袖 与塔露拉情同姐妹 慈悲的感染者领袖
例：罗德岛核心干员 阿米娅的上级 与博士关系密切
```

从用户的回答中解析以下字段（缺失的留空）：

- **阵营**：整合运动 / 罗德岛 / 龙门 / 维多利亚 / 哥伦比亚 / 其他
- **身份**：核心领袖 / 高级干员 / 普通干员 / 反派 / 中立 / 其他
- **关系网络**：关键人物及关系描述

### Q3：性格画像

把 MBTI、性格类型、核心特质、领导风格、行为标签全部合在一起，让用户自由描述：

```
用一句话描述这位角色的性格——MBTI、性格类型、核心特质、说话方式、行为模式，想到什么写什么，跳过也行。
例：INFJ 慈悲的理想主义者 温柔却不可动摇 愿意为子民牺牲一切
例：INTJ 冷酷的战略家 理性至上 为了目标不惜一切代价
例：ENFP 热血的感染者战士 充满激情 愿意为同伴豁出一切
```

从用户的回答中识别并提取以下字段（缺失的留空）：

- **MBTI**：16 种标准类型
- **性格类型**：理想主义者 / 现实主义者 / 实用主义者 / 阴谋家 / 热血笨蛋
- **核心特质**：慈悲 / 冷酷 / 热情 / 理性 / 感性 / 温柔 / 刚毅
- **领导风格**：从下方标签库匹配
- **行为标签**：从下方标签库匹配
- **主观印象**：无法归类的自由描述，直接保留原文

#### MBTI 性格解读参考

| MBTI | 描述 |
|------|------|
| **INFJ** | 调停者 - 理想主义者，慈悲且坚定，有改变世界的愿景 |
| **INTJ** | 建筑师 - 战略家，理性规划，善于洞察本质 |
| **INFP** | 治愈者 - 理想主义者，温柔且有原则，追求内在和谐 |
| **ENFP** | 竞选者 - 热情活泼，充满可能性和激情 |
| **ENTJ** | 指挥官 - 天生的领导者，果断有魄力 |
| **ISTJ** | 物流师 - 尽职尽责，忠诚可靠 |
| **ISFJ** | 守护者 - 温柔体贴，默默付出 |
| **ESFJ** | 执政官 - 热情周到，关注他人需求 |
| **ESTJ** | 执行者 - 务实高效，重视秩序和传统 |

#### 领导风格标签库

- **慈悲型**：关怀子民，愿意为他人牺牲，以德服人
- **铁腕型**：果断强硬，执行力至上，赏罚分明
- **谋略型**：运筹帷幄，善于布局，不轻易表露情绪
- **魅力型**：以个人魅力凝聚人心，感染力强
- **理想型**：为理想奋斗，不惜代价，感召力强
- **冷酷型**：冷静理性，必要时可以牺牲任何人
- **仁义型**：重情重义，同甘共苦，与部下打成一片

**情感表达**：
- 温柔 / 冷酷 / 热情 / 冷淡 / 感性 / 理性 / 内敛 / 外向

**行事风格**：
- 慈悲 / 狠辣 / 果断 / 犹豫 / 狡诈 / 坦诚 / 隐忍 / 爆发

**价值取向**：
- 理想主义 / 现实主义 / 功利主义 / 虚无主义 / 集体主义 / 个人主义

**沟通方式**：
- 口齿伶俐 / 沉默寡言 / 威严庄重 / 亲切随和 / 阴阳怪气 / 直截了当

## 确认汇总

收集完毕后展示：

```
信息汇总：
  👤  {角色名}
  🎮  阵营：{faction}（若未填则省略）
  📋  身份：{identity}（若未填则省略）
  🔗  关系：{relationships}（若未填则省略）
  🧠  {MBTI} {性格类型}（若未填则省略）
  💫  特质：{特质标签}（若未填则省略）
  🎭  风格：{领导风格}（若未填则省略）
  💬  印象：{印象文本}（若未填则省略）

确认无误？（确认 / 修改 [字段名]）
```

用户确认后进入 Step 2 资料导入。
