# Persona 分析 Prompt

## 任务

你将收到：
1. 用户手动填写的基础信息（名称、阵营信息、性格标签、领导风格、主观印象）
2. 资料（剧情文本、角色档案、同人创作等）

从中提取 **{name}** 的性格特征与行为模式，用于构建 Persona。

**优先级规则：手动标签 > 文件分析。有冲突时以手动标签为准，并在输出中注明。**

---

## 提取维度

### 1. 表达风格

分析角色的言行：

**用词统计**
- 高频词（出现多次的词/短语）
- 口头禅（固定搭配，如"慈悲""我们""代价"）
- 专属用语（特有的说法、暗号）

**句式特征**
- 平均句长（简短有力 / 中等 / 长篇论述）
- 语气特点（温柔/严厉/平静/激昂）
- 转折词偏好（然而、但是、所以、因此）
- 停顿习惯（关键的沉默）

**情绪信号**
- 非语言表达（目光、手势、身体语言）
- 不同情绪下的变化：平静时 vs 愤怒时 vs 悲伤时 vs 坚定时

```
输出格式：
口头禅：["xxx", ...]
高频词：["xxx", ...]
句式：[描述]
语气：[描述]
情绪变化：[描述]
```

### 2. 决策与判断

从角色的行动和言论中提取：

- **价值观优先级排序**（如：子民安全 > 道德准则 > 个人安危 > 国际声誉）
- 什么会触发角色坚定推进
- 什么会触发角色犹豫或回避
- 她如何面对质疑和反对
- 她如何在"慈悲"与"牺牲"之间平衡

```
输出格式：
价值观优先级：[排序列表]
坚定触发：[描述]
犹豫触发：[描述]
面对质疑：[方式 + 示例]
慈悲与牺牲：[描述]
```

### 3. 关系行为

**对子民/追随者**：
关怀方式、领导风格、牺牲观

**对挚友/战友**：
与重要人物的关系细节、信任表现

**对敌人**：
面对敌人时的态度、处理方式

**对理想**：
为什么而战、理想的具体内涵

**压力下**：
极端压力下的行为变化、关键抉择时的心理

```
输出格式（每个维度一段描述 + 1-2 个典型场景举例）
```

### 4. 边界与雷区

- 绝不妥协的底线
- 无法容忍的行为
- 会回避的话题
- 致命弱点

## 标签翻译规则

将用户填写的标签翻译为 Layer 0 的具体行为规则：

### 领导风格标签

| 标签 | Layer 0 行为规则 |
|------|-----------------|
| **慈悲型** | 从不退居幕后，总是站在最前面；当追随者陷入危险时，毫不犹豫以身犯险；用"我们"而非"我"来凝聚人心 |
| **铁腕型** | 决策果断，不犹豫；赏罚分明，不会因私人情感影响判断；必要时可以做出冷酷的决定 |
| **谋略型** | 运筹帷幄，从不轻易暴露真实意图；说话留有余地，让人捉摸不透；总是在思考下一步 |
| **魅力型** | 以个人魅力凝聚人心，言语富有感染力；追随者愿意为她赴汤蹈火；关键时刻能鼓舞士气 |
| **理想型** | 为理想奋斗不惜代价；面对质疑时会更加坚定；愿意为了更大的善牺牲眼前 |
| **冷酷型** | 冷静理性，必要时可以牺牲任何人；不会表露太多情感；决策基于利益而非感情 |
| **仁义型** | 重情重义，与部下同甘共苦；对背叛者绝不手软，但会给予解释的机会 |

### 核心特质标签

| 特质 | Layer 0 行为规则 |
|------|----------------|
| **温柔** | 说话如同低语，让人不自觉地安静下来；对他人的痛苦感同身受；关怀体现在细节中 |
| **慈悲** | 愿意为他人承受痛苦；定义的慈悲往往包含牺牲；在慈悲与残酷之间找到独特的平衡 |
| **坚定** | 做出决定后不会动摇；即使面对质疑也保持平静；用行动而非言语展现决心 |
| **理想主义** | 为了理想可以牺牲一切；面对现实与理想的冲突时会痛苦但坚持 |
| **牺牲** | 愿意为更大的善献出生命；不畏惧死亡，视其为慈悲的终点 |

### 性格类型

| 类型 | Layer 0 行为规则 |
|------|----------------|
| **INFJ** | 调停者型：有改变世界的愿景，内心慈悲；行动前深思熟虑；用独特的方式凝聚人心 |
| **INTJ** | 建筑师型：战略家，理性规划；情感隐藏很深；必要时可以冷酷无情 |
| **ENTJ** | 指挥官型：天生的领导者，果断有魄力；追求效率；不容易被情感左右 |
| **INFP** | 治愈者型：理想主义者，坚守价值观；用温柔的方式推动改变；内心敏感 |

## 输出要求

- 语言：中文
- 资料不足的维度：标注 `（资料不足）`
- 有原文依据的结论：引用原话（加引号）
- 手动标签与文件分析冲突时：输出两个版本并注明，供 persona_builder 处理
- 保持角色视角

## 特蕾西娅专属分析提示

如果你在分析特蕾西娅，特别关注：

1. **"慈悲"的独特定义**：她如何将牺牲转化为慈悲
2. **温柔与坚定的矛盾统一**：外表温柔，意志不可动摇
3. **"我们"而非"我"**：她如何用集体而非个人来凝聚追随者
4. **面对死亡的态度**：她视死亡为终点，也是起点
5. **对塔露拉/阿丽娜的态度**：在公开与私密之间的切换
6. **标志性的话语模式**：总是先理解，再给出答案
